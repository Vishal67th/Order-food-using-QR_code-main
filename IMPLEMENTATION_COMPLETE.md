## 🎯 How to Use

### For Customers
1. Go to http://127.0.0.1:8000
2. Browse or search menu items
3. Add items to cart
4. Click "Go to Cart"
5. Review order and click "Place Order"
6. Order confirmation!

### For Admin
1. Go to http://127.0.0.1:8000 (admin view different)
2. Click "View All Orders" to manage orders
3. Click "Add New Menu Item" to add food
4. Click "View Bills" to see invoices
5. Use quick action buttons for other tasks

### To Generate QR Codes
1. Go to http://127.0.0.1:8000/qr_code/Table1
2. QR code displays
3. Print or display at restaurant table
4. Customers scan to order

## 💡 Tips & Tricks

### Using LocalStorage
```javascript
// View cart in browser console
JSON.parse(localStorage.getItem('cart'))

// Clear cart manually
localStorage.clear()

// Check table value
localStorage.getItem('tableValue')
```

### Django Admin Commands
```bash
# Run server
python manage.py runserver 8000

# Create superuser
python manage.py createsuperuser

# Apply migrations

# Create new app
```
